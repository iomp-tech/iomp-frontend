### Онлайн-институт психологии - Institute of Modern Psychology

## Стек технологий

* ReactJs + Hook
* react-router-dom
* Redux
* Redux thunk
* node-sass
* axios

## Установка
Клонируйте репозиторий.
```
git clone https://github.com/iomp-tech/iomp-frontend.git
```

Переход в рабочий каталог.
```
cd iomp-frontend
```

Установка зависимостей, запуск сборки.
```
npm install
npm start
```
